## 2. Schema confirmation

Schema confirmed against `year=1951/month=01`:
```text
time           datetime64[us]
station        str   (WIGOS-style: "0-20000-0-06210")
stationname    str
lat, lon       float64
height         float64
FH             float64   (hourly mean wind speed; not the thesis variable)
FX             float64   (hourly maximum wind gust; primary variable)
```

Station ids in the parquet use the WIGOS form `0-20000-0-NNNNN`. The 5-digit tail `NNNNN` matches the `station_id` column in `data/station_metadata.csv`. The normaliser is `data_loading.station_id_to_5digit`.
